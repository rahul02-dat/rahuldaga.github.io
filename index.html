PROJECT / 002

NL-SQL — Natural Language → SQL Agent

95.34% query accuracy  ·  15+ security rules

Agentic · Security
  • Locally-running agentic pipeline translating plain-English questions into safe, read-only SQL using a Qwen-coder (20B) LLM via Ollama — no external API, fully air-gapped.
  • Multi-layer security gate enforces 15+ rules before any query is generated: input validation, SQL injection detection, prompt injection blocking, and operation policy (INSERT/UPDATE/DELETE/DROP are rejected entirely).
  • Three-agent pipeline: Intent Agent classifies queries (VALID / INCOMPLETE / DISALLOWED), SQL Agent generates schema-aware SELECT statements from live schema introspection, Insight Agent converts raw results into plain-English analysis.
  • Self-healing mechanism — if a generated query errors at runtime, the LLM automatically attempts a corrected fix before surfacing the failure to the user.
  • Achieved 95.34% query accuracy across diverse natural language styles; full timestamped security audit trail and structured insight logs.
Python SQLite Ollama (Qwen-coder) LLM Agents SQL Security

PROJECT / 003

PromptWall — LLM Security Gateway

89.34% detection  ·  7,000+ labeled samples

LLM Security
  • Multi-layered security gateway intercepting untrusted inputs (URLs, documents, user text) before they reach an LLM agent — covering prompt injection, data exfiltration, agentic intent abuse, and HOUYI-style context manipulation.
  • 8+ specialized detectors run in parallel: IntentClassifier, HOUYIPatternDetector, SemanticThreatDetector (embedding similarity), PromptInjectionDetector, AgenticIntentDetector, ExfiltrationDetector, HiddenContentAnalyzer, and ContentDeobfuscator (Base64/hex/URL, up to 3 layers deep).
  • Weighted risk scoring produces one of four decisions — ALLOW / SANITIZE / REQUIRE_APPROVAL / BLOCK — with thresholds fully configurable in YAML; critical safety overrides always fire regardless of score.
  • Optional ML override layer (RandomForest / XGBoost) trained on labeled samples; OCR content analyzer (Tesseract) extracts and inspects text embedded inside uploaded images.
  • Evaluated on 7,000+ labeled prompt samples achieving 89.34% attack detection with low false-positive rates on legitimate inputs.
Python Scikit-Learn · XGBoost sentence-transformers Tesseract OCR PyYAML
